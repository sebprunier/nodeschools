Bienvenu dans le workshop NodeSchool pour apprendre ou approfondir la programmation en JavaScript sur NodeJS.

Pré-requis : Installation de Node.JS

Vous trouverez ici les installeurs de NodeJS pour windows mac et linux. Une fois installés, vous aurez accès à 2 outils en ligne de commande :

  • node, l'interpréteur javascript côté serveur
  • npm, le gestionnaire de package node (Node Package Manager)

Ateliers à la carte

Chaque atelier vous guidera dans une série d'exercices en ligne de commande. Les exercices ont un problème à résoudre, des conseils pédagogiques pour y arriver, et une vérification automatisé du résultat. Les ateliers proviennent de http://nodeschool.io

  • Learn You The Node.js For Much Win!
  • ExpressWorks
  • Stream Adventure
  • Functional JavaScript
  • Promise It Won't Hurt
  • Async You

Learn You The Node.js For Much Win!

Apprenez les bases de node : entrées/sorties asynchrone, serveur http basique, ...

Installation :

npm --registry http://192.168.0.105:8080 install -g learnyounode

En cas de problème sous mac/linux, ajoutez sudo devant.

Vous avez désormais une commande learnyounode. Lancez juste learnyounode pour faire l'atelier !
